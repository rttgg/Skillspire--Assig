Higher-Order Function

Functions are values, all simple functions can be compose together using higher order function. Just like string, numbers…functions can be assigned to variables or passing to higher order function composition.
Higher order function good for composition. One function put in other function allows us to compose a lot of small functions into bigger higher order function.
Higher order methods are

  • For Each
  • Filter
  • Map
  • Sort
  • Reduce
--- Let’s look at filter method
Filter method allows us to do just filter things out from the array. Filter will loop through each item in the array and for each item it’s going to pass it into the callback function and when it does it expect the function to return either true or false.
--- Let’s look at Map method
Map method instead of filtering things out we can create new arrays of anything from a current array.
Map expect the call back to return to any object. We can use it to create completely new objects.
--- Let’s look at reduce
Reduce method is the multi-use, Reduce is a function on the array object and just like map and filter it takes a callback function but unlike map and filter it wants an object, the object will use it as a starting point for the function to be passed as the first argument to reduce method.
